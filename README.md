# AutoReport LB - Vehicle Transparency Platform

A MERN stack application for neutral, append-only vehicle history reporting in Lebanon.

## Prerequisites

- Node.js
- MongoDB (running locally or cloud)
- Google Gemini API Key

## Setup

### 1. Backend

```bash
cd server
npm install
cp .env.example .env
# Edit .env and add your MONGO_URI and GEMINI_API_KEY
npm run seed # (Optional) Populates DB with sample data
node server.js
```

### 2. Frontend

```bash
cd client
npm install
npm run dev
```

## Features

- **Public Search**: Search by VIN or Plate Number.
- **Timeline**: View reported events chronologically.
- **AI Summary**: Neutral summary generated by Gemini 1.5 Pro.
- **Partner Portal**: Insurance companies can login and report events.
- **Append-Only**: Events cannot be edited or deleted once submitted.
